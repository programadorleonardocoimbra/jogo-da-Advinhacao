# 🎯 Jogo: Adivinhe o Nome Correto

***Um pequeno projeto em Java que simula um jogo simples de adivinhação no terminal. O usuário deve escolher o nome correto entre várias opções para ganhar.***

### 📌 Sobre o Projeto

***Este projeto foi desenvolvido com o objetivo de praticar conceitos básicos da linguagem Java, como:***

- Entrada de dados com Scanner
- Estruturas de repetição (while)
- Condicionais (if)
- Comparação de Strings
- Interação com o usuário via terminal

***O jogo apresenta uma lista de nomes e desafia o usuário a descobrir qual é o nome correto.***

### 🎮 Como Funciona
#### O sistema exibe uma lista de nomes:
- Leandra
- Michele
- Lucia
- Sara
- Bruna

## O usuário deve digitar um nome.
### O programa verifica:
#### ✅ Se o nome estiver correto → o usuário vence
#### ❌ Caso contrário → o usuário pode tentar novamente
#### O jogo continua até o acerto.
## 🖥️ Exemplo de Execução
#### ======================================
#### ==== JOGO: ADVINHE O NOME CORRETO ====
#### ======================================

((( Leandra - Michele - Lucia - Sara - Bruna )))

 - Digite aqui: Maria
 - Você Perdeu :( - TENTE NOVAMENTE
 - Digite aqui: Lucia
 - Você acertou!
 - Fim de Jogo
### 🚀 Tecnologias Utilizadas
 - Java
 - JDK 8+
 - Terminal / Console
### 📂 Estrutura do Código
 - Scanner: Captura a entrada do usuário
 - while (true): Mantém o jogo rodando até o acerto
 - if: Valida a resposta do usuário
 - equals() e equalsIgnoreCase(): Comparação de Strings
